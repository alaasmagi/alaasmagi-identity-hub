## Translation Review

The Estonian and Finnish resource files are complete and key-aligned with English, but the following categories should receive human review before production release:

- Identity and consent workflow wording, especially messages around access approval, revoked access, consent expiration, and invited-client access.
- Admin terminology for clients, roles, allowed origins, security events, registration type, and client secrets.
- Two-factor authentication wording for authenticator apps, recovery codes, remembered browsers, and QR-code setup.
- Validation and status messages that include security-sensitive account state.

No individual key was intentionally left untranslated; English fallback is available through the default resource behavior if a future translation key is missing.
